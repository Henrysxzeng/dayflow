# FlowCast · V2.0 产品路线图

**制定时间：** 2026年5月  
**最后更新：** 2026年5月  

---

## 已完成功能（备案等待期间完成）

### ✅ Bug 修复
- 时长选项增加0.5小时步进（0.5h~6h）
- 跳过按钮移至输入框顶部，不再被键盘遮挡

### ✅ 核心体验优化
- **3步沉浸式引导页**（痛点→价值→长期承诺）
- **任务编辑功能**（··· 按钮 → 编辑/删除，目前最大UX漏洞修复）
- **任务模板库**（4类20个模板，add-task 一键应用）
- **番茄钟持久化**（云端记录状态，App 重启后恢复倒计时）

### ✅ 数据系统
- **情绪追踪**（完成任务后4选1，第一章解锁后显示趋势图）
- **分层解锁系统**（3章，基于行为质量防刷）
  - 第1章：5天有效活跃 → 拖延初步分析 + 情绪趋势
  - 第2章：12天 + 5次用时反馈 → 时间校准 + 能量曲线
  - 第3章：22天 + Streak历史≥7 → 完整洞察 + Wrapped + 心流达人

### ✅ 好友系统（完整版）
- **好友码互加**（6位字母+数字，不依赖微信关系链，无需认证）
- **好友协作任务**（新建任务时邀请好友，双方完成+默契度+2）
- **默契度5段制**（初识投缘→志趣相投→灵犀暗通→默契十足→心流共振）
- **共同任务历史**（"历史"Tab 查看合作记录）
- **编辑名片**（昵称+头像emoji，让好友认出你）
- **好友挑战裂变**（完成庆祝页"挑战朋友"按钮，分享带邀请的小程序卡）

### ✅ 界面重组
- "我的"Tab 分为"统计"和"设置"两个子Tab，解决信息过载
- 新增第4个底部 Tab："好友"
- 个人成长路径三章全部展示解锁内容预览

## 数据驱动决策（上线后验证）

| 假设 | 目标数值 | 验证方式 |
|------|---------|---------|
| 用户愿意每天打开 | Day7 留存 > 40% | 数据库日志统计 |
| AI 计划真实有用 | 完成率 > 60% | daily_logs 统计 |
| Streak 机制有效 | 平均连续天数 > 5 | users.streak 统计 |
| 好友功能促进留存 | 有好友用户留存 > 无好友用户 +20% | 分组对比 |
| 付费意愿存在 | Streak>30天用户中 > 20% 愿付费 | 用户调研 |
