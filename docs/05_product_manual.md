# FlowCast · 产品功能手册

> 让每天的任务像水流一样被引导，进入心流状态。

**版本：** V2.0  
**更新时间：** 2026年5月

---

## 这是什么？

FlowCast 是一款 AI 驱动的每日任务规划微信小程序，同时也是一个可以和好友共同打卡的效率社交工具。

核心价值：**不是帮你管理所有任务，而是每天早上帮你选出最该做的几件事，告诉你几点做、为什么现在做。**

---

## 功能全览

### 一、AI 智能日程规划

**1. 智能每日计划**
- 选择今天可用时长（0.5小时步进，从0.5h到6h）
- 可选：用自然语言描述今天的固定安排，如"上午10-11点开会，下午3点接孩子"
- AI 根据人类能量曲线安排最优时段：
  - 早上8-12点（前额叶最活跃）→ 最难、最重要的任务
  - 中午12-13:30 → 自动空出，不排任何任务
  - 下午13:30-15点（低谷期）→ 轻松事务（回邮件、整理）
  - 下午15-17:30（第二高效期）→ 中等难度任务
- 每个任务附带科学依据注释，如：
  - "前额叶峰值，复杂分析效率最高"
  - "蔡格尼克效应：先啃硬骨头，完成后负担消失"
  - "帕金森定律：今天不动，明天更难动"
- 忙碌时段显示为红色分隔线，午饭自动出现

**2. 随手清空列表**
预计 ≤10 分钟的碎片任务自动分流，不占主计划认知资源，单独列在"随手清空"。

**3. 重新生成计划**
- 顶部"X小时 修改 ›" → 选新时长重新生成
- 底部"↺ 重新生成计划" → 用当前时长直接重排，无弹窗

---

### 二、任务管理

**4. 任务录入**
- 名称、截止日期（精确到几点几分）、预计用时、重要程度
- 预计用时支持自定义输入（分钟数）
- 截止日期精确到时间后，AI 排期更准确

**5. 任务编辑**
- 任务清单页，每张卡片右侧 `···` 按钮 → 弹出"编辑/删除"
- 编辑会预填原有内容，修改后保存

**6. 任务模板库**
新建任务页顶部"从模板快速添加"，4个分类共20个常用模板：
- 📚 学习：阅读书籍、英语学习、整理笔记、复习知识点、在线课程
- 💼 工作：写周报、回复邮件、整理文件、准备会议材料、项目汇报
- 🏃 健康：冥想放松、散步、健身运动、拉伸运动、睡前准备
- 🏠 生活：购物采买、整理房间、处理账单、做饭、联系家人

**7. 智能结转**
未完成任务不需要手动处理，夜间系统自动识别并结转到明日候选池。

---

### 三、专注工具

**8. 番茄钟专注模式**
- 任务卡片右侧 🍅 专注 → 全屏专注模式
- 25分钟倒计时，App 切后台计时不停（时间戳计算，不依赖 interval）
- **持久化**：App 被系统杀死后重新打开，自动恢复倒计时
- 结束后震动提醒，可选"休息5分钟"或"提前完成"

### 四、坚持系统

**9. Streak 连续天数**
- 每天完成计划 Streak +1
- 免死金牌 🃏：昨天断了，今天用一张补救（默认1张）
- 今日休息 🌿：没有任务时标记休息日，Streak 不断

**10. 成就系统（7个里程碑）**

| 成就 | 触发 |
|------|------|
| 🌱 起步 | 完成第1个任务 |
| 🎯 十件成就 | 累计完成10件 |
| 💪 效率达人 | 累计完成50件 |
| 🚀 百件英雄 | 累计完成100件 |
| 🔥 初燃 | 连续3天 |
| ⚡ 一周战将 | 连续7天 |
| 🏆 月度传奇 | 连续30天 |

### 五、个人成长路径（分层解锁）

基于真实使用质量解锁，无法通过刷任务数量触发。

| 章节 | 解锁条件 | 解锁内容 |
|------|---------|---------|
| 🔒 第一章：起步 | 5天有真实完成记录 | 拖延规律初步分析、情绪与效率趋势图、完整成就系统 |
| 🔒 第二章：深入 | 12天使用 + 5次记录实际用时 | 时间校准分析、专属能量高峰时段、深度每周AI总结 |
| 🔒 第三章：传奇 | 22天使用 + Streak最高达7天 | 完整个人洞察报告、月度Wrapped、🌊 心流达人徽章、推送个性化 |

**每章都展示解锁后内容预览，用户知道坚持的价值。**

---

### 六、时间校准分析

每次勾选完成任务后，Flow 问实际花了多久（4选1）：⚡ 比预估少 / ✅ 差不多 / ⏰ 多了一些 / 😅 超了很多

**积累10次解锁**：Flow 计算你的平均偏差系数，之后生成计划时自动修正——如果你通常低估30%，Flow 会自动给任务加30%时间。

未解锁时显示"X/10 个数据点"进度条，本身也是留存钩子。

---

### 七、情绪追踪

所有任务完成后，情绪选择（4选1）：⚡ 精力充沛 / 😊 还不错 / 😴 有点疲 / 😩 很疲惫

**第一章解锁后**在"我的→统计"显示近期情绪趋势图，看心情与效率的关系。

---

### 八、推送系统

**11. 晨间推送**（每天08:00）
完成今日任务后授权，第二天早上收到推送 → 点击确认今日可用时长 → 生成计划。

**12. 晚间再激活**（每天20:00）
当天任务未完成时推送："今天还有X件事没完成，现在还来得及。"

**13. 周五预载**（每周五18:00）
引导填写下周最重要的事，周一生成计划时优先安排。

---

### 九、失败标签系统

任务没做完，第二天早上 Flow 逐一询问原因（一键选择）：

- 😓 太难了 → 连续2次后自动拆解为更小子任务
- ⏰ 没时间 → 直接结转，提高次日权重
- 😶 不想做 → 降低优先级，本周末重新评估

---

### 十、好友系统（完整版）

**14. 好友码**
每个用户有唯一的6位好友码（如 FC3K7M），可复制或分享给朋友。朋友输入码 → 双方成为好友。

**15. 好友列表与默契度**

好友之间通过共同完成任务积累默契度：

| 等级 | 分数 | 称号 | 副标题 |
|------|------|------|--------|
| 🌱 | 0-10 | 初识投缘 | 入门·初见好感 |
| 🔥 | 11-30 | 志趣相投 | 初阶·彼此合拍 |
| ✨ | 31-60 | 灵犀暗通 | 中阶·渐有感应 |
| 💎 | 61-100 | 默契十足 | 高阶·高度契合 |
| 🌊 | 100+ | 心流共振 | 满级·极致同频 |

双方完成同一个共同任务 → 默契度 +2。好友列表按默契度排序。

**16. 好友协作任务**
新建任务时开启"邀请好友一起完成"，选择好友 → 任务同步发送。

好友在"好友 → 邀请"Tab 看到邀请 → 接受 → 任务自动进入各自的任务池。

**17. 共同任务历史**
"好友 → 历史"Tab 查看与每位好友合作记录，显示双方完成状态和获得的默契度。

**18. 编辑名片**
自定义昵称（最多12字）和头像 emoji，让好友认出你。

### 十一、社区功能

**19. 社区月度挑战**
每月一个全社区匿名挑战（如"五月：连续14天完成计划"）。所有人参与，看到总参与人数和你的进度。两人条件（天数+任务数）满足任一即可，每天更新进度。

**20. 每周 AI 总结**
每周 Flow 根据你的数据生成个性化总结，缓存后自动展示，不重复调用 AI。

### 十二、数据与分享

**21. 七日完成率趋势图**
固定7个柱状图，无数据的天显示"-"，休息日显示绿色。

**22. 分享今日成绩**
完成所有任务后，庆祝页一键分享成绩卡（canvas 生成）：
```
今天，我赢了。
3 件任务已完成 · 共 140 分钟
🔥 连续高效第 12 天

```

**23. 好友挑战裂变**
完成庆祝页同时显示"挑战朋友"按钮，分享当月挑战卡，好友点击进入 FlowCast。

**24. 分享本月成绩**
"我的→统计"页面一键分享本月累计数据。

### 十三、页面结构

**底部 Tab（4个）：**
- 今日：AI 计划 + 番茄钟 + 完成庆祝
- 任务：任务清单（带编辑/删除）+ 新建任务（含模板+好友邀请）
- 好友：好友列表/邀请/历史 + 默契度 + 名片编辑
- 我的：统计子Tab（分析、图表、成长路径）/ 设置子Tab（成就、挑战、AI风格、隐私）

## 使用指南

### 第一次打开

3步引导动画：
1. 痛点共鸣："任务一堆，不知道先做哪个？"
2. 产品价值：Flow 如何帮你规划+排期
3. 长期价值：用得越久，Flow 越懂你

然后点"加第一个任务，开始吧"进入新建任务页。

### 日常使用流程

```
早上收到推送 → 确认今日时长（0.5h步进）
→ 可选：填写时间约束 → AI 生成计划（5-15秒）
→ 执行（可开番茄钟）→ 勾选完成 → 标记实际用时
→ 全部完成 → 选今日情绪 → 庆祝页 → 授权明天推送
                                      ↘ 分享成绩或挑战好友
```

### 和好友一起用

```
自己：好友Tab → 复制/分享好友码
好友：输入码 → 成为好友（默契从🌱开始）
新建任务 → 开启"邀请好友" → 选好友 → 发出邀请
好友：接受邀请 → 任务加入各自计划
双方完成 → 默契度 +2 → 历史记录更新
```

## 常见问题

**Q: AI 生成计划要多久？**
A: 通常5-15秒，网络好时更快。失败时有"重试"弹窗，不会直接报错。

**Q: 我的好友码在哪里？**
A: 点底部"好友"Tab，名片区域就能看到，可以复制或分享。

**Q: 默契度怎么涨？**
A: 你和好友一起加入同一个任务，两人都完成后自动 +2。

**Q: 数据会不会丢失？**
A: 所有数据存储在腾讯云，删除小程序或换手机都不丢，重新打开继续。

**Q: 体验版和正式版的数据是一样的吗？**
A: 是的，共用同一个云数据库，数据按用户微信 ID 隔离，不会混。

**Q: Streak 断了怎么办？**
A: 每人默认有1张免死金牌，昨天断了今天打开会弹出补救选项。
