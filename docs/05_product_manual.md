# DayFlow · 产品功能手册

> 让每天的任务像水流一样被引导，进入心流状态。

---

## 这是什么？

DayFlow 是一款 AI 驱动的每日任务规划微信小程序。

很多人的痛点不是"不知道要做什么"，而是"有太多事不知道先做哪个"，于是陷入焦虑、拖延、什么都没做的循环。

DayFlow 每天早上问你一个问题："今天能用几小时？"，然后根据你的任务清单和时间，帮你选出今天最该做的 3-5 件事，告诉你几点做什么。

---

## 核心功能

### 1. 智能每日计划

- 告诉 Flow 今天有几小时可用
- 可选：填写今天的时间安排（"上午10-11点开会，下午3点接孩子"）
- Flow 根据**人类能量曲线**安排任务：
  - 早上精力最好的时段 → 安排难度最高、最重要的任务
  - 午饭时间（12:00-13:30）自动空出
  - 下午低谷期 → 安排轻松事务（回邮件、整理）
  - 下午第二个高效期 → 安排中等难度任务
- 每个任务都有 AI 生成的科学注释（如"前额叶峰值，复杂分析效率最高"）

### 2. 随手清空列表

预计 10 分钟以内的碎片任务（买东西、回消息）不进入主计划，单独列在"随手清空"里，不占用认知资源。

### 3. 番茄钟专注模式

点任务卡片右侧的 🍅 专注 → 进入 25 分钟全屏专注模式 → 结束后可选休息 5 分钟或直接标为完成。App 切到后台计时不停。

### 4. Streak 连续天数

每天完成计划，连续天数 +1。担心断掉？有两个保障：
- **免死金牌**：昨天没打开，今天有机会用金牌补救
- **今日休息**：今天没有任务，标记休息日，Streak 照样涨

### 5. 晨间推送

完成今天的计划后，可以授权"明天早上 8 点的推送"。第二天你会收到：

> "Flow 早！今天有 3 件要事。今天能挤出几个小时？"

点击直接进入计划确认页面。

### 6. 失败原因标签

任务没做完没关系，第二天早上 Flow 会问"昨天那件事没做，是太难了、没时间、还是不想做？"。Flow 会根据你的回答调整下次的安排——比如连续两次"太难了"，会自动把任务拆成更小的子任务。

### 7. 成就系统

| 成就 | 触发条件 |
|------|---------|
| 🌱 起步 | 完成第 1 个任务 |
| 🎯 十件成就 | 累计完成 10 件 |
| 💪 效率达人 | 累计完成 50 件 |
| 🚀 百件英雄 | 累计完成 100 件 |
| 🔥 初燃 | 连续 3 天 |
| ⚡ 一周战将 | 连续 7 天 |
| 🏆 月度传奇 | 连续 30 天 |

### 8. 时间校准分析

每次完成任务后，可以标记实际用时（比预估少/差不多/多了一些/超了很多）。积累 10 次后解锁：Flow 会自动分析你的时间估算习惯，并在生成计划时校正——如果你通常低估 30%，Flow 会自动给任务加 30% 的时间。

### 9. 社区月度挑战

每月有一个全社区挑战（如"五月：连续14天完成计划"）。所有人匿名参与，可以看到参与人数和你当前的进度。

### 10. 每周 AI 总结

每周 Flow 会根据你的数据生成一段个性化总结，类似"这周你完成了 23 件事，最高效的一天是周二，继续保持这个节奏"。

### 11. 分享成绩

完成今日计划后，可以生成并分享一张成绩卡给朋友：

```
今天，我赢了。
3 件任务已完成 · 共 140 分钟
🔥 连续高效第 12 天
Flow 陪你高效地活着
```

## 使用指南

### 第一次使用

1. 打开小程序
2. 点"加第一个任务"进入任务录入页
3. 填写：任务名称、截止日期（可精确到几点）、预计用时、重要程度
4. 回到首页，选择今天能用的时间
5. 可选：告诉 Flow 今天的时间安排
6. 等待 5-15 秒，AI 生成你的今日计划

### 日常使用流程

```
早上收到推送 → 打开 App → 确认今天可用时间
→ 查看 AI 计划 → 开始执行任务 → 逐一勾选完成
→ 全部完成 → 庆祝页 → 授权明天推送
```

### 录入任务技巧

- **预计用时 ≤ 10 分钟**：自动进入"随手清空"列表，不影响主计划
- **截止日期**：越近的任务权重越高，AI 会优先安排
- **重要程度**：配合截止日期构成四象限优先级

### 提高计划质量

点选时长后，在"告诉 Flow 你的时间安排"框里填入：
```
上午9-11点开会，下午4点有个面试
```
Flow 会自动绕开这些时段，把任务塞进空档，还会自动留午饭时间。

## 常见问题

**Q: AI 生成计划要多久？**
A: 通常 5-15 秒，网络好的情况下更快。

**Q: 计划生成了但我临时有变怎么办？**
A: 点顶部"X小时 修改 ›" 重新选时长 → 重新生成；或点底部"↺ 重新生成计划"用相同时长重新规划。

**Q: 任务没做完怎么办？**
A: 不用手动处理。夜间 Flow 会自动把未完成的任务加回明天的候选池，第二天早上会问你没做完的原因。

**Q: 今天什么任务都没有怎么办？**
A: 首页会显示"今日休息"按钮，点一下 Streak 照样涨，不会断。

**Q: 删掉小程序数据会怎样？**
A: 你的所有数据存储在云端，删除小程序或换手机不影响数据，重新打开就能继续。
