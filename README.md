# FlowCast · AI 每日任务调度助手

> 让每天的任务像水流一样被引导，进入心流状态。

微信小程序 | AI驱动 | 个人效率 + 好友协作

---

## 产品简介

FlowCast 是一款基于 DeepSeek AI 的每日任务规划微信小程序。核心理念：**不是帮你管理所有任务，而是每天早上帮你选出最该做的几件事，根据人类能量曲线安排到最合适的时间段。**

同时支持好友协作：邀请好友一起完成任务，积累默契度（初识投缘 → 志趣相投 → 灵犀暗通 → 默契十足 → 心流共振）。

## 核心功能（24项）

**AI 规划**
- AI 智能日程（人类能量曲线排期，科学注释）
- 时间约束输入（自然语言，0.5h步进）
- 动态难度校准（读取失败历史，调整计划强度）
- 时间校准分析（10次用时反馈后解锁AI系数修正）

**任务管理**
- 任务录入（截止时间精确到分钟）
- 任务编辑（··· 按钮 → 编辑/删除）
- 任务模板库（4类20个常用模板，一键填入）
- 随手清空（≤10分钟碎片任务自动分流）
- 智能结转（夜间自动处理未完成任务）

**专注工具**
- 番茄钟（25分钟，后台不停，App 重启自动恢复）

**坚持系统**
- Streak 连续天数 + 免死金牌 + 今日休息
- 成就系统（7个里程碑）
- 分层解锁路径（第1/2/3章，防刷设计）
- 情绪追踪（完成任务后标记心情，解锁趋势图）

**推送体系（4种）**
- 晨间AI计划推送（每天07:50，自动生成今日计划并推送）
- 晚间未完成提醒（每天20:00，还有未完成任务时推送）
- 周五下周预载（每周五18:00，回顾本周+规划下周）
- 逐任务截止倒计时 🆕（每30分钟扫描，精确到分钟的截止提醒）
- 每日首次打开自动弹订阅授权

**好友系统**
- 好友码互加（6位字母+数字，无需微信权限）
- 好友协作任务（邀请好友一起完成，双方完成+默契度）
- 默契度系统（5段：初识投缘→志趣相投→灵犀暗通→默契十足→心流共振）
- 共同任务历史
- 编辑名片（昵称 + 头像 emoji）

**数据与分享**
- 7日完成率趋势图
- 每周 AI 总结
- 社区月度挑战
- 分享今日成绩卡（canvas 生成）
- 好友挑战裂变（一键分享当月挑战）

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | 微信小程序原生（WXML/WXSS/JS） |
| 后端 | 腾讯云开发 CloudBase（Serverless） |
| 数据库 | 云开发 CloudDB（MongoDB，10个集合） |
