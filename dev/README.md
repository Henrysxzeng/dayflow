## 核心功能

**AI 规划** — AI 根据人类能量曲线排期，支持自然语言时间约束、动态难度校准、时间校准分析

**任务管理** — 截止时间精确到分钟、20个常用模板、碎片任务自动分流、智能夜间结转

**番茄钟** — 25分钟专注、后台计时不停、App 重启自动恢复

**坚持系统** — Streak 连续天数、免死金牌、7项成就、分层解锁、情绪追踪

**推送体系（4种）** — 晨间AI计划推送、晚间未完成提醒、周五下周预载、逐任务截止倒计时提醒

**好友协作** — 好友码互加、默契度5段成长、共同任务、分享成绩卡、社区月度挑战
